# Delta SkillPack v2 - Claude Code Plugin

This plugin provides multi-engine AI workflow orchestration capabilities.

## Available Skills

| Skill | Engine | Description |
|-------|--------|-------------|
| `delta:plan` | Claude Opus 4.5 | Generate implementation plans with deep reasoning |
| `delta:implement` | Codex GPT-5.2 | Execute code implementation with full-auto mode |
| `delta:review` | Claude Opus 4.5 | Perform thorough code review |
| `delta:ui` | Gemini 3 Pro | Design UI/UX specifications |
| `delta:ralph-init` | - | Initialize PRD from task description |
| `delta:ralph-start` | Multi-engine | Start autonomous development loop |
| `delta:ralph-status` | - | Check Ralph execution status |

## When to Use These Skills

### Planning Tasks (delta:plan)
Use when:
- Implementing new features requiring architectural decisions
- Refactoring existing code
- Need multiple implementation approaches to compare

### Implementation Tasks (delta:implement)
Use when:
- Have a clear plan ready to execute
- Need code changes across multiple files
- Implementing features based on approved plans

### Review Tasks (delta:review)
Use when:
- Implementation is complete
- Before merging or committing
- Need quality assurance

### UI Tasks (delta:ui)
Use when:
- Designing user interfaces
- Creating visual specifications
- Frontend component design

### Ralph Autonomous Loop (delta:ralph-start)
Use when:
- Have a complex task with multiple stories
- Need end-to-end automated development
- PRD-driven development workflow

## Commands

- `/ralph-loop "task"` - Start autonomous development loop
- `/cancel-ralph` - Cancel running Ralph loop

## Integration Notes

This plugin integrates with:
- obra-superpowers (brainstorming, debugging, TDD)
- Claude Code hooks system
- Git for version control

All skills output to `.skillpack/runs/<run_id>/` directory.
